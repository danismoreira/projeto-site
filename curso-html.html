Curso de HTML5 e CSS3

Pagína contruída a partir das orientações do curso em vídeo de Git e Github, do professor Gustavo Guanabara.

A HTML é uma linguagem de marcação para criação de sites. Atualmente na versão 5, ela usa a semântica dos elementos para criar o conteúdo de uma página Web. Com as CSS, cria-se as confgurações visuais que o site terá ("Gustavo Guanabara").
