Curso de JavaScprit

Pagína contruída a partir das orientações do curso em vídeo de Git e Github, do professor Gustavo Guanabara.

JavaScprit é uma linguagem de scrpit criada para aumentar a intereatividade entre um site e seus visitantes. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM ("Gustavo Guanabara").
